Arbres de décision¶

Dans cette tape, on ralise l'implmentation de l'algorithme de construction d'un arbre de dcision.

Entropie et gain d'information¶

On note $\{c_1, c_2,..., c_k\}$, l'ensemble des classes possibles.

Dans le cours, la mesure de dsordre prsente utilise, pour toute valeur $c$ de la classe le ratio $\frac{|Y(c)|}{|Y|}$ qui correspond la probabilit de la classe $c$ parmi $Y$. On note dans ce qui suit $p_1$ la probabilit de la classe $c_1$ parmi $Y$, etc.

Ainsi, on a $P=(p_1, p_2,..., p_k)$ la distribution de probabilits sur les classes ainsi dfinies.

L'entropie de Shannon de la distribution $P$ est donne par:

$ H_S(P) = -\sum_{i=1}^{k} p_i \log_k(p_i)$

Le logarithme utilis est le logarithme en base $k$ afin d'obtenir une valeur normalise de l'entropie.

Remarque: avec les notations vues en cours, on a $D(Y) = H_S(P)$.

Pour ce qui suit, une distribution de probabilits est reprsente par une liste Python $P=[p_1,...,p_k]$ de valeurs relles telle que

  • pour tout $i=1,..., k$ : $0 \leq p_i \leq 1$
  • $\sum_{i=1}^{k}p_i = 1$

Discrétisation des attributs numériques¶

Comme on l'a vu en cours, discrtiser un attribut numrique consiste en trouver un seuil de coupure dans son ensemble de valeurs.

Par exemple, soit l'attribut Age contenant les valeurs 17, 18, 20, 27, 35, 37, 40. On discrtise cet attribut en 2 valeurs en prenant, par exemple, le seuil 30: cela permet de sparer d'un ct les valeurs infrieures ou gales 30, et de l'autre les valeurs strictement suprieures 30.

Le seuil est choisi afin d'optimiser la distribution des classes des exemples qui se trouvent de chaque ct du seuil. Une fois un seuil choisi, on calcule le dsordre des classes associes aux exemples de chaque ct du seuil et on agrge les 2 valeurs de dsordre obtenues pour dterminer la qualit de la coupure l'aide de la mesure $Q$ vue en cours.

Soit $X$ un ensemble d'exemples, et soit $A$ un attribut que l'on souhaite discrtiser. Pour cela, on constitue l'ensemble des valeurs prises par les exemples de $X$ pour l'attribut $A$. Soit $A_X$ cet ensemble de valeurs.

La discrtisation de $A_X$ s'effectue par les tapes suivantes:

  • Etape 1: ordonner de faon croissante les valeurs de $A_X$;
  • Etape 2: regarder tous les seuils possibles de coupure entre 2 valeurs conscutives de $A_X$ (on prend comme seuil de coupure la valeur mdiane entre ces 2 valeurs). Chaque seuil trouv cr 2 intervalles qui gnrent 2 sous-ensembles de $X$: l'ensemble $X_1$ des exemples dont la valeur pour $A$ et infrieure ou gale au seuil, et l'ensemble $X_2$ des exemples dont la valeur pour $A$ et suprieure au seuil. On dtermine la qualit du seuil par la mesure de qualit d'un test vue en cours. On note $Q(seuil)$ cette valeur.

La meilleure coupure pour discrtiser $A_X$ est celle qui minimise la valeur $Q(seuil)$.

Par exemple, soit $X=\{17, 18, 20, 27, 35, 37, 40\}$, chaque valeur tant associes aux classes +1, +1, -1, -1, +1, +1, -1 respectivement. Un seuil de coupure possible est $31$ ($31 = \frac{27+35}{2}$). Ce seuil coupe $X$ en $X_1=\{17, 18, 20, 27\}$ et $X_2=\{35, 37, 40\}$.

  • pour $X_1$, la probabilit de chaque classe est: $p(+1)= 0.5$ et $p(-1)=0.5$ donc, $H_S(X_1)=1$;
  • pour $X_2$, la probabilit de chaque classe est: $p(+1)= \frac{1}{3}$ et $p(-1)=\frac{2}{3}$ donc, $H_S(X_2)=0.9182958340544896$;
  • on a donc: $$Q(seuil=31) = \frac{4}{7} H_S(X_1) + \frac{3}{7} H_S(X_2);$$ soit $Q(seuil=31) = 0.9182958340544896$

def discretise(LSet, col):
    """ LabelledSet * int -> tuple[float, float]
        Hypothse: LSet.size() >= 2
        col est le numro de colonne sur X  discrtiser
        rend la valeur de coupure qui minimise l'entropie ainsi que son entropie.
    """
    # initialisation:
    min_entropie = 1.1  # on met  une valeur max car on veut minimiser
    min_seuil = 0.0     
    # trie des valeurs:
    ind= np.argsort(LSet.x,axis=0)
    
    # calcul des distributions des classes pour E1 et E2:
    inf_plus  = 0               # nombre de +1 dans E1
    inf_moins = 0               # nombre de -1 dans E1
    sup_plus  = 0               # nombre de +1 dans E2
    sup_moins = 0               # nombre de -1 dans E2       
    # remarque: au dpart on considre que E1 est vide et donc E2 correspond  E. 
    # Ainsi inf_plus et inf_moins valent 0. Il reste  calculer sup_plus et sup_moins 
    # dans E.
    for j in range(0,LSet.size()):
        if (LSet.getY(j) == -1):
            sup_moins += 1
        else:
            sup_plus += 1
    nb_total = (sup_plus + sup_moins) # nombre d'exemples total dans E
    
    # parcours pour trouver le meilleur seuil:
    for i in range(len(LSet.x)-1):
        v_ind_i = ind[i]   #vecteur d'indices
        courant = LSet.getX(v_ind_i[col])[col]
        lookahead = LSet.getX(ind[i+1][col])[col]
        val_seuil = (courant + lookahead) / 2.0;
        # M-A-J de la distrib. des classes:
        # pour rduire les traitements: on retire un exemple de E2 et on le place
        # dans E1, c'est ainsi que l'on dplace donc le seuil de coupure.
        if LSet.getY(ind[i][col])[0] == -1:
            inf_moins += 1
            sup_moins -= 1
        else:
            inf_plus += 1
            sup_plus -= 1
        # calcul de la distribution des classes de chaque ct du seuil:
        nb_inf = (inf_moins + inf_plus)*1.0     #rem: on en fait un float pour viter
        nb_sup = (sup_moins + sup_plus)*1.0     # que ce soit une division entire.
        # calcul de l'entropie de la coupure
        val_entropie_inf = shannon([inf_moins / nb_inf, inf_plus  / nb_inf])
        val_entropie_sup = shannon([sup_moins / nb_sup, sup_plus  / nb_sup])
        val_entropie = (nb_inf / nb_total) * val_entropie_inf \
                       + (nb_sup / nb_total) * val_entropie_sup
        # si cette coupure minimise l'entropie, on mmorise ce seuil et son entropie:
        if (min_entropie > val_entropie):
            min_entropie = val_entropie
            min_seuil = val_seuil
    return (min_seuil, min_entropie)

import graphviz as gv

class ArbreBinaire:
    def __init__(self):
        self.attribut = None   # numro de l'attribut
        self.seuil = None
        self.inferieur = None # ArbreBinaire Gauche (valeurs <= au seuil)
        self.superieur = None # ArbreBinaire Gauche (valeurs > au seuil)
        self.classe = None # Classe si c'est une feuille: -1 ou +1
        
    def est_feuille(self):
        """ rend True si l'arbre est une feuille """
        return self.seuil == None
    
    def ajoute_fils(self,ABinf,ABsup,att,seuil):
        """ ABinf, ABsup: 2 arbres binaires
            att: numro d'attribut
            seuil: valeur de seuil
        """
        self.attribut = att
        self.seuil = seuil
        self.inferieur = ABinf
        self.superieur = ABsup
    
    def ajoute_feuille(self,classe):
        """ classe: -1 ou + 1
        """
        self.classe = classe
        
    def classifie(self,exemple):
        """ exemple : numpy.array
            rend la classe de l'exemple: +1 ou -1
        """
        if self.est_feuille():
            return self.classe
        if exemple[self.attribut] <= self.seuil:
            return self.inferieur.classifie(exemple)
        return self.superieur.classifie(exemple)
    
    def to_graph(self, g, prefixe='A'):
        """ construit une reprsentation de l'arbre pour pouvoir
            l'afficher
        """
        if self.est_feuille():
            g.node(prefixe,str(self.classe),shape='box')
        else:
            g.node(prefixe, str(self.attribut))
            self.inferieur.to_graph(g,prefixe+"g")
            self.superieur.to_graph(g,prefixe+"d")
            g.edge(prefixe,prefixe+"g", '<='+ str(self.seuil))
            g.edge(prefixe,prefixe+"d", '>'+ str(self.seuil))
        
        return g

Construction de l'arbre de décision¶

Pour construire un arbre de dcision, il faut raliser les tapes suivantes:

  • si l'entropie de Shannon de la distribution des classes de l'ensemble courant est infrieure epsilon (rel positif fix et qui est un paramtre de l'algorithme), alors construire une feuille avec cet ensemble, la classe associe cette feuille est la classe majoritaire;
  • sinon, pour chaque attribut, calculer le seuil de discrtisation et l'entropie de Shannon correspondante
    • choisir l'attribut qui donne une valeur d'entropie minimale
    • l'attribut choisi sert constituer un noeud de l'arbre de dcision, les arcs issus de ce noeud sont labeller par un test (infrieur ou gal ou suprieur) par rapport au seuil trouv

Classification avec arbre de décision¶

Pour classer un nouvel exemple avec un arbre de dcision, il faut parcourir l'arbre, de la racine vers une feuille en respectant les tests rencontrs dans chaque arc. Une fois atteint une feuille, la classe de l'exemple est celle prsente dans la feuille.

class ArbreDecision(cl.Classifier):
    # Constructeur
    def __init__(self,epsilon):
        #valeur seuil d'entropie pour arrter la construction
        self.epsilon= epsilon
        self.racine = None
    
    # Permet de calculer la prediction sur x => renvoie un score
    def predict(self,x):
        # classification de l'exemple x avec l'arbre de dcision
        # on rend 0 (classe -1) ou 1 (classe 1)
        classe = self.racine.classifie(x)
        if (classe == 1):
            return(1)
        else:
            return(-1)
    
    # Permet d'entrainer le modele sur un ensemble de donnes
    def train(self,set):
        # construction de l'arbre de dcision 
        self.set=set
        self.racine = construit_AD(set,self.epsilon)

        gtree = gv.Digraph(format='png')
